\doxysubsection{Detailed Description}
The execption that is thrown when the value of a nonexistent entry is requested. 

Note that this exception is only thrown when the value of an \mbox{\hyperlink{classini_1_1Entry}{Entry}} is obtained; not when the entry is obtained from a \mbox{\hyperlink{classini_1_1Section}{Section}}.
